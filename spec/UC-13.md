## UC-13 · Integrate with remote-controlled video player mpv

**Actor:** User  
**Goal:** Open the current video with remote-controlled mpv via command `:view`

---

### Flow

1. User positions the cursor on a video file.
2. User invokes `:view` command.
3. mex checks the file extension; if it is not a known video type an error is shown.
4. mex builds the absolute path (`target_root + target_path`).
5. mex connects to the mpv IPC socket (`/tmp/mex-mpv.sock`) and sends a `loadfile` command.
   If mpv is not running, mex spawns it with `--idle=yes` and waits for the socket to appear.
6. On error (file missing, `mpv` not found, socket timeout) a status message is shown.

### Multimedia keyboard controls

Once a video is open in mpv, the multimedia keys on the keyboard control playback directly
from mex without leaving the TUI:

| Key | Action |
|-----|--------|
| ⏯ Play / Pause | Toggle play/pause (`cycle pause`) |
| ⏹ Stop | Stop playback (mpv stays alive in idle mode) |
| ⏭ Next Track | Move cursor to next video in the list and open it |
| ⏮ Prev Track | Move cursor to previous video in the list and open it |

### Notes

- `:view` only works on known video extensions (mp4, mkv, avi, mov, webm, flv, wmv, m4v, ts, m2ts, mpeg, mpg, ogv, 3gp, divx, rmvb); other files show an error.
- Multi-selection is ignored; always opens the cursor file.
- The TUI is not suspended; mex remains fully interactive while mpv is open.
- Next/Prev skip non-video files and wrap around the filtered list.

### Implementation

- **`mex/src/player.rs`** — `trait RemoteController { open_file, stop, play_pause }` and
  `struct MpvController` implementing it via `std::os::unix::net::UnixStream` (no extra deps).
  Socket path: `/tmp/mex-mpv.sock` (constant `MPV_SOCKET`).
  mpv is spawned with `--idle=yes --input-ipc-server=<socket>`; socket polled 20 × 100 ms.
- **`mex/src/app.rs`** — `view_selected()`, `view_next_video()`, `view_prev_video()`,
  `mpv_play_pause()`, `mpv_stop()`; `"view"` added to `KNOWN_COMMANDS` and `execute_command`.
- **`mex/src/main.rs`** — `KeyCode::Media(…)` events dispatched to the above methods.
