## UC-06 · Create view of selection

**Actor:** User  
**Goal:** Create a filesystem-based view of the current selection (copying files is not an option, it should be instant)

**Preconditions:**

* We assume media files are on a Linux native filesystem, where hard linking is possible
* we need a configure directory path where those views can be materialized

**Main Flow:**
1. User filters files
2: command :vc <name> create a view in the filesystem
